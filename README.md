# 🌐 Meus Sites Favoritos

📌 **Objetivo do Projeto**  
Criar uma página web simples em **HTML5** para armazenar e acessar rapidamente meus sites favoritos.  
O projeto faz parte da atividade prática da disciplina **Sociedade da Informação e Comunicação**.

---

💫 **Sites Incluídos**
1. 🔎 **Google** – Ferramenta de busca essencial.  
2. 🎬 **YouTube** – Plataforma de vídeos e entretenimento.  
3. 📖 **Wikipedia** – Enciclopédia colaborativa online.  
4. 🐙 **GitHub** – Hospedagem de código e colaboração.  
5. 💡 **Stack Overflow** – Comunidade de perguntas e respostas para programadores.

---

🧠 **Tecnologias Utilizadas**
- HTML5  
- Estrutura básica de tags (`<h1>`, `<h2>`, `<img>`, `<a>`)  
- Organização de arquivos locais (HTML + imagens)  
- Links externos com `target="_blank"` para abrir em nova aba  

---

🌿 **Como funciona**
- O arquivo `meus-sites-favoritos.html` exibe o título **Meus sites favoritos: Alan**.  
- Cada site possui:
  - Nome destacado com `<h2>`  
  - Imagem representativa (`.png`)  
  - Link clicável para acessar o site
